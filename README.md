Mood-Based Movie Recommender Web App

Description:

This project is a web application that recommends movies based on the user's mood. Users can select their current mood, and the application fetches relevant movies using a public API. The app also allows users to search, filter, and sort movie results, providing an interactive and responsive experience.

Purpose:

The goal of this project is to demonstrate the use of JavaScript concepts such as API integration, array higher-order functions, and dynamic UI rendering.

API Used:

TMDB (The Movie Database) API

Features:
Mood-based movie recommendations
Search functionality
Filtering by genre or year
Sorting movies (rating/title)
Responsive UI

🧰 Technologies Used
HTML
CSS
JavaScript (Fetch API, Array HOFs)
